Key Findings

Finding 1 — Platform Is Fine; This Is Segment-Specific

The rest of Pulseboard's customer base grew +10.8% in the same window. The FinTech Pro drop is not seasonality and not a platform issue — it's two accounts diverging from a healthy platform trend.

Finding 2 — Event Types Tell Two Stories

Users who log in are doing more substantive work per session. The entire decline is in session-lifecycle events (login/logout) and API calls — not in dashboarding and file activity.

Caveats & Limitations
What to keep in mind when acting on this analysis:
  • n = 2 accounts. No statistical generalization is possible. These are account-level descriptions.
  • API discontinuation cause unknown. ZGarcia's api_call=0 is consistent with deliberate discontinuation but could also reflect a silent credential expiry. CS must verify.
  • Window sensitivity. The −29.7% headline is accurate for May 7–Jun 6. On a per-day normalized basis across a shifted window, the rate difference is −6%. The headline magnitude is partly window-dependent.
  • Event data covers only 92 days (Mar 7–Jun 6, 2025). No prior-year comparison is possible.
  • report_view increase in PJohnson is from a low base (2→5 events). Directionally positive; not large in absolute terms.
